  • ✓Subir um Hermes em Docker num VPS, conectar Telegram e GitHub

A ideia central

O Hermes não é um chatbot que você configura uma vez e esquece. É um colega de trabalho que precisa ser treinado: corrigido quando erra, ensinado quando repete, separado quando cresce.

O segredo não é configurar tudo de uma vez. É usar, corrigir, salvar aprendizados e evoluir aos poucos. Este curso te leva por esse caminho.
